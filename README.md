Installing plugins
Before we can start our environment, lets install the hostmanager plugin:

     $ vagrant plugin install vagrant-hostmanager


List plugins

     $vagrant plugin list
     
     
Vagrant Up

    $vagrant up
    
Vagrant status

    $vagrant status
    
Vagrant restart

    $vagrant reload
